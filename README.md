# PHPA_star

PHPA_star (PHPA\*) ist eine Hoch paraalelisierte Variante des Hierachical Pathfinding A_Star (HPA\*) Algorithmus.

Grundansatz ist die parralelisierung über die GPU oder Massive Parralelisierung über verteielte systeme.
Dies soll eine Möglichkeit bieten auf graphen wegfindung effizent zuberechnen.
Für das Verfahren wird eine aufwändige Vorverarbeitung verwendet welche mit einem Gegebenen Clustering-Algorithmus die Knoten in Gruppen einteilt welche später als Seperate Teilprobleme verarbeitet werden können.
